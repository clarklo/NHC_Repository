加選項目

骨質密度：X光式檢查腰椎及大腿骨股股質密度，有無骨質疏鬆或股質減少 。

頸動脈超音波：頸動脈血流及血管掃瞄，是否有血管狹窄 。

甲狀腺超音波：甲狀腺結構掃瞄，甲狀腺節結、腫大、癌症 。

女性癌病篩檢：婦科超音波掃瞄、CA-125、CA-153，卵巢癌、乳癌腫瘤標記及子宮、卵巢、陰道、膀胱疾病檢查 。

乳房超音波：乳房超音波結構檢查，是否有乳癌、纖維囊腫或其他乳房病變 。

乳房攝影：X光乳房攝影，是否有乳癌、纖維囊腫或其他乳房病變 。

24小時心電圖：可攜帶式24小時連續心電圖，檢查心律不整、心肌缺氧 。

心臟超音波、運動心電圖：心臟超音波掃描及履帶式跑步心電圖，檢查心臟結構與功能、是否有缺氧的情況 。

內視鏡檢查：1. 無痛胃鏡，2. 無痛大腸鏡，3. 無痛胃加大腸鏡，專科麻醉下進行內視鏡檢查，胃鏡：食道、 胃及十二指腸發炎、潰瘍及腫瘤大腸鏡:痔瘡、大腸息肉、發炎及大常病兆檢查 。

電腦斷層檢查 。

  • 1.腹部電腦斷層：肝、膽、胰、胃腸、腎臟、脾臟掃瞄，肝、膽、胰、胃腸、腎臟、脾臟系統病變 。
  • 2.電腦斷層冠狀動脈鈣化指數：檢查心臟冠狀動脈鈣化指數，表示冠狀動脈硬化程度及危險程度 。
  • 3.冠狀動脈鈣化指數及肺癌篩檢：冠狀動脈鈣化程度及肺部結構掃瞄，評估心血管疾病風險及篩檢肺癌及肺病 。
  • 4.電腦斷層虛擬腸鏡：利用3D重組，模擬大腸鏡檢查，大腸息肉、發炎及大腸癌檢查 。
  • 5.64排電腦斷層冠狀動脈掃瞄：利用3D重組，檢視冠狀動脈血管，檢查冠狀動脈硬化指數及血管狹窄程度 。

3T 全景磁振造影檢查 。

  • 1.腦神經血管磁振攝影造影檢查：腦部結構、頸動脈、顱內血管、頸椎掃描，是否曾中風、腦瘤、腦部病變、血管阻塞 。
  • 2.癌症篩檢磁振造影：頭、胸、腹、骨盆、及脊椎等八大部位，偵測頭頸部、胸部、腹部、骨盆腔、及脊椎疾病與腫瘤 。
  • 3.全身磁振造影：腦部結構、頸動脈、顱內血管、頸椎、胸、腹、骨盆、及脊椎等部位，偵測腦部是否曾中風、腫瘤、 腦部病變、血管阻塞等、胸部、腹腔是否有疾病與腫瘤及脊椎疾病與腫瘤篩檢 。
  • 4.乳房磁振造影：乳房磁振造影結構檢查，是否有乳癌、纖維囊種或其他乳房病變 。
  • 5.攝護腺磁振造影：攝護腺磁振造影結構檢查，是否有攝護腺癌、鈣化或其他攝護腺病變 。
  • 6.脊椎磁振造影檢查(單一部位)：脊椎結構磁振造影掃描(單一頸椎、胸椎或腰椎)，檢查腰椎疾病、椎間盤凸出、脊椎滑脫等 。
  • 7.全椎磁振造影檢查：脊椎結構磁振造影掃描，檢查腰椎疾病、椎間盤凸出、脊椎滑脫等 。
